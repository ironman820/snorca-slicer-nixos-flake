# Code of Conduct

## Our standard

Participation in this project should stay respectful, constructive, and focused on improving the repository.

Examples of expected behavior:

- giving actionable technical feedback
- discussing tradeoffs without personal attacks
- keeping issues and reviews relevant to the project
- assuming good intent

Examples of unacceptable behavior:

- harassment or abusive language
- discriminatory behavior or language
- deliberate disruption of discussion or review
- sharing private information without permission

## Enforcement

Project maintainers may remove comments, issues, or contributions that do not align with this code of conduct.

## Scope

This code of conduct applies to repository discussions, issues, pull requests, and related project spaces.
