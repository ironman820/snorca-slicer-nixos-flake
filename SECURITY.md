# Security Policy

## Supported versions

This repository tracks packaging for the latest Orca Slicer release selected through the included update workflow.

## Reporting a vulnerability

If you believe there is a security issue in this repository's packaging or update process, avoid posting a full public exploit immediately.

Instead:

1. Open an issue with a minimal description of the problem.
1. Ask for a private follow-up channel if sensitive details are required.
1. Include affected files, impact, and reproduction steps.

For vulnerabilities in Orca Slicer itself, report them to the upstream Orca Slicer project through its security or issue process.
